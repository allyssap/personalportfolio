Command Line Hangman

About

This command line hangman game was a fun idea I challenged myself with. It had been a couple years since I had programmed in python and I wanted to test my problem solving ability and see how fast I could relearn python.

I created a simple command line hangman game that prompts the user through each step, as well as takes a new random word from a .txt file every game. After the player has won, or run out of turns, the player can choose to play again or end the program.

Goals and Accomplishments

With this project I hoped to build a simple project that showed my ability to pick up skills quickly and to work in a timely manner. I think that I did a great job creating neat, well commented code especially considering the time constraints I had.

I gave myself 3 hours to design and code the game. I ended up with a 10 turn hangman command line game. If you would like to try it, make sure you have python installed on your computer and you can find the repo here.
